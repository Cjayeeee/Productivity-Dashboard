# Productivity Dashboard

A modern productivity dashboard built using HTML, CSS, and JavaScript.

## Features

- Real-time clock and date
- Current weather information
- Task management
- Daily motivational quotes
- Local storage support
- Responsive design
- Clean and modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- QuotableAPI
- WeatherAPI


## Installation

1. Clone the repository

```bash
git clone https://github.com/Cjayeeee/Productivity-Dashboard.git
```

2. Open the project folder

```bash
cd Productivity-Dashboard
```

3. Run the project using Live Server or open `index.html`.

## API Setup

1. Create an account at WeatherAPI.
2. Generate an API key.
3. Replace the API key in the JavaScript file:

```js
const apiKey = "YOUR_API_KEY";
```
